Binary Magician

   by Yunus E.V. / yunusemrevurgun@hotmail.com


Binary magician is a mostly browser based program (written in HTML,CSS,JavaScript,SQL,C++) that creates SQL files with table values converted into binary (ASCII/UTF-8). It works simple! Give us your desired table, column and value information and we will do the magic!

*input format for writing data into SQL file: "new_table_name"   ","   "new_column_name"   ","   "new_value_for_column" 
example -----> table1,column1,Hello World!
example 2 ----> table_name,column_name,my phone number is: +1 1111 111,my name is Example Name,I live in planet earth
*NOTE: in every creation a new table, a new column and a new value(s) are created. For creating multiple values for the same table and column, see example 2 above.
*NOTE: in the current version of our program, you can not write into the same table or column again. Every time you run the program, a new table with a new column and new values will be created.
*NOTE: you can not create more than 1 table or column with the same name. Also note that maximum characters per table/coulmn/value (including spaces in values) is 20 characters. Also don't create more than 20 values for a column.
